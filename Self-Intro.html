What is HTML ?


HTML is a mark up language which is used specifically for designing webpages, these are some thoughts of mine on HTML and CSS till now. Different opinions are welcomed :)
It's somehow like a natural language
- code lines are highly readable, even to someone with zero knowledge in HTML
- attributes, styles and elements are named based on relevant English words, it's easy to remember and use

It's a tag based language
- tags shows where the position of the elements are, also the types of the elements.
- the critical part is to set up the "tag structure" of the webpage, after that, writing content is just like filling the blanks

CSS feels like a block designing tool
- the webpages are basically a LEGO, constructed block by block, CSS set up a design rules for each block and assign to the blocks by using id and classes
- HTML generate the content, CSS is in respond for the layout
